## Install

```sh
npm install
```

## Usage

```sh
npm start
```

## Directory Structure

    | .env                     # Environment variables
    | app.js                   # App entry point
    | package-lock.json
    | package.json
    ├── bin
    ├── models                 # Database models
    ├── node_modules
    ├── routes                 # Express route controllers for all the endpoints of the app
    ├── services               # The business logic is here
    ├── views
    └── README.md
